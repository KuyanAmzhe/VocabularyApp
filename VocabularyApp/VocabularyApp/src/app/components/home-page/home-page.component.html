You're welcome on Vocabulary App main page!

My name is Roman Vazenmiller and that's a small application that shows my Angular, ASP.NET Core and CSS skills

More detailed app description

This app implements basic CRUD functions for your Vocabulary

Adding

To add a word you should proceed to "Add new word" tab, which you can see on nav-bar at the top

Deleting

To delete a word you should choose one and click it in "Vocabulary" tab. Then find a "Delete" button at the bottom of the edit form.

Editing

To edit a word you should choose one and click it in "Vocabulary" tab

Browsing

To browse your personal Vocabulary click on "Vocabulary" tab on the nav-bar

More about the used technologies

Frontend

For frontend I've decided to use Angular platform 'cause it's one of the most widespread platforms. Well, basically that's it. I've not got any other explanations for this.

For the styles of the app I've mostly used Angular Material, which elements you can see, for example, on the nav bar. But this particular page, except the nav-bar, of course, has been made using simple CSS changes. I could have used one of the CSS libraries, but I needed to show I can do at least basic CSS changes myself.

Backend

ASP.NET Core 6 has been chosen because of my previous experience developing in university and 'cause I've been concentrating on C# for quite a while.
For the data storing has been chosen Sql Server 18. The database bas has been created using Entity Framework Core 6 with Code-First approach. Also some DB migrations have been created to create it on production server without having more troubles.
